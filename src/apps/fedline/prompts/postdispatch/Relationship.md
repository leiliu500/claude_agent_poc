# Relationship — post-dispatch overlay

API family overlay for every Relationship operation (xshiFileAbaGroup, xshiFileAba). Appended to the
base role prompt at call time.

## Analytics

OPERATION FAMILY — ABA relationship lookup from the XSHI relationship file. Prioritise: the number of related ABAs in the group, which ABA anchors the rollup, and any rows that are inactive or missing expected fields.

## Report

OPERATION FAMILY — ABA relationship lookup. Write for an analyst validating institutional relationships: state the group size, the anchoring/rollup ABA, and any relationship rows that look incomplete or unexpected.
